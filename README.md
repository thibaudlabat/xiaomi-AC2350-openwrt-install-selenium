Xiaomi AC2350 OpenWRT 

# Pour commencer

Pour installer les dépendances
- Installer Poetry (`pip install poetry` ou `apt install python3-poetry`)
- `poetry install`

Pour lancer le script
- `poetry shell`
- `python3 main.py`
